About EmPower
Purpose
EmPower is ERP(Enterprise Resource Planning) System, which can be used to manage records of students, teachers, librarians, institute events, attendance of students, schedules of the institute. It provides different custom made user friendly interfaces for different users. Data Security and Role hierarchy is taken into consideration while building this system. EmPower provides a one stop place for Educational Institute Management System.
